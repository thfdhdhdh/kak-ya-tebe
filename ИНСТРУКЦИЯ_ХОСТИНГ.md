# Инструкция по размещению сайта на бесплатном хостинге

## Вариант 1: Beget (Рекомендуется для России)

1. **Регистрация:**
   - Перейдите на https://beget.com/ru
   - Нажмите "Регистрация"
   - Укажите номер телефона для подтверждения

2. **Создание сайта:**
   - Войдите в панель управления
   - Создайте новый сайт (бесплатный поддомен вида `ваш-логин.beget.app`)

3. **Загрузка файлов:**
   - Откройте файловый менеджер в панели управления
   - Загрузите файлы из архива `site.zip`:
     - `index.html` → в корневую папку сайта (обычно `public_html`)
     - `style.css` → в корневую папку сайта
     - Все файлы из папки `images` → в папку `images` на сервере

4. **Готово!** Ваш сайт будет доступен по адресу `ваш-логин.beget.app`

## Вариант 2: Sprinthost

1. **Регистрация:**
   - Перейдите на https://sprinthost.ru
   - Зарегистрируйтесь через Яндекс, Google или ВКонтакте

2. **Создание сайта:**
   - В панели управления создайте новый сайт
   - Получите бесплатный поддомен

3. **Загрузка файлов:**
   - Используйте FTP или файловый менеджер
   - Загрузите все файлы в папку `public_html` или `www`

1. **Регистрация:**

2. **Создание сайта:**

3. **Загрузка файлов:**
   - Используйте файловый менеджер
   - Загрузите файлы в корневую директорию сайта

## Вариант 4: GitHub Pages (если доступен)

1. **Создайте репозиторий на GitHub:**
   - Зарегистрируйтесь на github.com
   - Создайте новый репозиторий (например, `my-site`)

2. **Загрузите файлы:**
   - Распакуйте `site.zip`
   - Загрузите все файлы в репозиторий через веб-интерфейс или Git

3. **Включите GitHub Pages:**
   - Settings → Pages
   - Source: выберите `main` branch
   - Ваш сайт будет доступен по адресу `ваш-логин.github.io/название-репозитория`

## Структура файлов на сервере:

```
/
├── index.html
├── style.css
└── images/
    ├── photo1.jpg
    ├── photo2.jpg
    ├── photo3.jpg
    ├── photo4.jpg
    ├── photo5.jpg
    ├── photo6.jpg
    ├── photo7.jpg
    └── photo8.jpg
```

**Важно:** Убедитесь, что папка `images` находится в той же директории, что и `index.html`!

## Быстрая загрузка через FTP:

Если хостинг предоставляет FTP доступ:

1. Используйте FTP-клиент (FileZilla, WinSCP)
2. Подключитесь к серверу (данные в панели управления)
3. Загрузите файлы в папку `public_html` или `www`
