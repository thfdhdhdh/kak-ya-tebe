# Сайт "Как я тебе?"

## Структура проекта

- `index.html` - основная HTML страница
- `style.css` - стили сайта
- `images/` - папка для фотографий

## Установка

1. Поместите ваши 9 фотографий в папку `images/` с именами:
   - `photo1.jpg`
   - `photo2.jpg`
   - `photo3.jpg`
   - `photo4.jpg`
   - `photo5.jpg`
   - `photo6.jpg`
   - `photo7.jpg`
   - `photo8.jpg`
   - `photo9.jpg`

2. Откройте `index.html` в браузере

## Особенности

- Диагональный фон с двумя оттенками розового
- Текст "Как я тебе?" в стилизованной рамке
- Сетка 3x3 для фотографий
- Адаптивный дизайн для мобильных устройств
- Эффекты при наведении на фотографии
